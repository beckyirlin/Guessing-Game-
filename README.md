# Guessing Game 
This website is inspired by apps such as Solitaire Cash, where you can bet money to test your skills on strategy and accuracy. The technology used involves creating several files, each showcasing a specific part of the code that enables the website to function. Additionally, the project's deployment involved using the modern hosting platform called Render. The link to the actual website is: https://guessing-game-nm1y.onrender.com. One technical challenge was withdrawing the cash and integrating it into the work. One area that could be improved is the use of the money, specifically the possibility of betting money on whether you can answer a puzzle faster than the other person.
